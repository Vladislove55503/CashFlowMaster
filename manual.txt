Адрес репозитория: https://github.com/Vladislove55503/CashFlowMaster

В целях экономии времени я не стал усложнять проект конфигурационным файлом 
в каталоге с settings.py, где хранился бы секретный ключ 
и переменные для базы данных (если бы была postgres).

Инструкция для linux (на основе Debian, как Ubuntu и Mint) и python 3.10+

1) Создайте пустой каталог и в него клонируйте репозиторий. 
   Сюда же установите виртуальное окружение командой "python3 -m venv venv".
   
2) Запустите вирт. окр. командой "source venv/bin/activate".
   Перейдите в каталог "CashFlowMaster" командой "cd CashFlowMaster".
   Установите зависимости командой "pip install -r requirements.txt".

3) Перейдите в следующий каталог "CashFlowMaster" командой "cd CashFlowMaster".
   Запустите тестовый сервер командой "python manage.py runserver".
   Перейдите по ссылке в терминале ("http://127.0.0.1:8000/").

Готово. Пользуйтесь.

Note: Возможно, вы заметили, что в зависимостях уже есть django-debug-toolbar, 
      поэтому вам не придётся устанавливать его самостоятельно. 
      Чтобы он заработал, нужно в settings.py установить DEBUG=True.
